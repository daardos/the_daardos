🛠️ О сайте

Технические детали, история версий и технологии, на которых построен этот проект.

🔧 Технологический стек

  • HTML5, CSS3 (Flexbox, Grid, Custom Properties)
  • JavaScript (ES6+) без фреймворков
  • Canvas API для интерактивных элементов
  • Google Fonts (Inter, Roboto, Fira Code)
  • Formspree для обработки форм
  • GitHub Pages для хостинга
  • LocalStorage для сохранения настроек

✨ Возможности

  • Мультиязычность (RU / EN)
  • Тёмная / светлая тема
  • Сохранение настроек без сервера
  • Интерактивная карта навыков
  • Калькулятор стоимости (в планах)
  • Виджет «Поделиться»
  • Модальное окно с бонусом
