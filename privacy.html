Политика конфиденциальности

Дата последнего обновления: 26 мая 2026 г.

Настоящая Политика конфиденциальности объясняет, как я (далее – «владелец сайта», «я») собираю, использую и защищаю персональные данные, которые вы предоставляете при использовании данного сайта-портфолио.

1. Какие данные собираются

При заполнении контактной формы (имя, email, сообщение) эти данные передаются через сервис Formspree на мою электронную почту. Я не храню их в базе данных сайта, так как её нет. Также хостинг GitHub Pages может автоматически собирать техническую информацию (IP-адрес, тип браузера, User-Agent) в целях безопасности и работоспособности. Эти данные обрабатываются согласно политике конфиденциальности GitHub и недоступны мне как владельцу сайта.

2. Как используются данные

Предоставленная информация используется исключительно для связи с вами по вашему запросу (обсуждение проекта, ответы на вопросы). Данные не передаются третьим лицам, не используются для рекламы или спама.

3. Файлы cookie

Сайт не использует рекламные или аналитические куки. Настройки внешнего вида (тема, шрифт, язык) сохраняются в локальном хранилище вашего браузера (localStorage) и не передаются на сервер. Вы можете очистить эти данные в настройках браузера в любое время, это не нарушит работу сайта.

4. Безопасность

Я принимаю разумные меры для защиты передаваемых данных. Форма отправляется по защищённому протоколу HTTPS. Сервис Formspree также обеспечивает безопасность передачи. Однако помните, что передача данных через интернет не может быть гарантированно защищена на 100%.

5. Сторонние сервисы

Для работы контактной формы используется Formspree (formspree.io). Пожалуйста, ознакомьтесь с их политикой конфиденциальности. Сайт размещён на GitHub Pages, который может собирать анонимную статистику посещений в соответствии с политикой GitHub.

6. Ваши права

Вы имеете право запросить удаление предоставленных вами данных из моей почты. Для этого свяжитесь со мной через контактную форму или email. Я отвечу в течение 7 рабочих дней.

7. Изменения в политике

Я могу обновлять этот документ. Новая версия вступает в силу с момента публикации. Рекомендую периодически проверять эту страницу.
