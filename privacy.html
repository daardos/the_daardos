Политика конфиденциальности

Дата последнего обновления: 21 мая 2026 г.

Настоящая Политика конфиденциальности объясняет, как я (далее — «владелец сайта», «я») собираю, использую и защищаю информацию, которую вы предоставляете при использовании данного сайта-портфолио.

1. Какие данные собираются

При заполнении контактной формы (имя, email, сообщение) эти данные передаются через сервис Formspree на мою электронную почту. Я не храню их в базе данных, так как сайт является статическим. Также хостинг GitHub Pages может автоматически собирать техническую информацию (IP-адрес, тип браузера, User-Agent) в целях безопасности и работоспособности. Эти данные обрабатываются согласно политике конфиденциальности GitHub и недоступны мне как владельцу сайта.

2. Как используются данные

Предоставленная информация используется исключительно для связи с вами по вашему запросу (обсуждение проекта, ответы на вопросы). Данные не передаются третьим лицам, не используются для рекламы или спама.

3. Хранение настроек

Сайт использует локальное хранилище браузера (localStorage) для сохранения ваших персональных настроек: тема оформления, язык, шрифт, акцентный цвет. Эти данные хранятся только на вашем устройстве и не передаются на сервер. Вы можете очистить их в любой момент через настройки браузера. Сайт не использует файлы cookie для отслеживания или рекламы.

4. Безопасность

Я принимаю разумные меры для защиты передаваемых данных. Контактная форма отправляется по защищённому протоколу HTTPS. Сервис Formspree также обеспечивает безопасность передачи. Однако абсолютной гарантии безопасности при передаче данных через интернет не существует.

5. Сторонние сервисы

Для работы контактной формы используется Formspree (formspree.io). Рекомендую ознакомиться с их политикой конфиденциальности. Сайт размещён на GitHub Pages, который может собирать анонимную статистику посещений в соответствии с политикой GitHub.

6. Ваши права

Вы имеете право запросить удаление предоставленных вами данных из моей почты. Для этого свяжитесь со мной через контактную форму или по email. Я отвечу в течение 7 рабочих дней.

7. Изменения в политике

Я могу обновлять этот документ. Новая версия вступает в силу с момента публикации. Рекомендую периодически проверять эту страницу.
